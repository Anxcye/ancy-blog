# Roadmap

## Vision
Build a clean, maintainable blog platform from scratch with `Go + PostgreSQL + Redis + Vue3`.

## Milestones
1. Foundation
- Repository scaffolding, coding rules, CI baseline, local dev environment.
2. Auth and Core Data
- Admin login, token flow, base schema, migrations.
3. Content Management
- Article/category/tag CRUD in admin.
4. Public Blog APIs
- Article list/detail, category/tag filters, site settings.
5. Interactions and Delivery
- Comments, upload, caching strategy, production deployment.

## Initial Scope (MVP)
- Single admin account authentication.
- Article publishing and reading.
- Categories and tags.
- Basic site settings.

## Out of Scope (for now)
- Multi-role RBAC.
- Multi-tenant support.
- Complex analytics pipeline.
